 ### 学習ログ

資格学習と個人開発を、測定・自動化・改善で運用した記録です。
日々の学習を計測可能にし、つまずきをデータで管理し、ツール化して効率を上げてきました。

### このログで示せること
- **学習の自動計測**：AppSheetで正答率・解答時間・誤答カテゴリを集計し日次通知する仕組みを自作
- **本番トラブル対応**：Heroku Postgres の EOL対応をバックアップ〜再デプロイまで手順化
- **アルゴリズム読解**：擬似言語を専用ツールで実行トレース
- **継続性**：2025/10〜の日次学習記録
